# 官方實價查詢截圖工具

這套工具的目的，是用官方「內政部不動產交易實價查詢服務網」像一般使用者一樣選條件、按搜尋，然後保留官方網頁截圖與擷取表格。截圖可作為主要依據，CSV/JSON 只作為輔助整理。

## 使用方式

1. 打開 `案件設定.json`，修改縣市、行政區、門牌/社區關鍵字、交易期間等條件。
2. 雙擊 `點我官方查詢截圖.bat`。
3. 第一次執行會安裝 Playwright，需要網路。
4. 完成後，到 `output/evidence/日期_案件名稱/` 查看輸出。

也可以準備多份設定檔（例如 `案件設定_某案.json`），用指令分別執行：

```
node scripts/official_lvr_evidence.js 案件設定_某案.json
```

## 會產生的檔案

- `001_官方查詢條件畫面.png`（已裁切到查詢條件列，清楚顯示你勾選的條件）
- `002_官方查詢結果第1頁.png`
- `官方查詢結果_全部.csv`（逐頁擷取全部結果，與官方畫面一致）
- `官方查詢結果_全部.json`
- `篩選後_排除房地車.csv`（`排除房地車` 設 true 時才產生，供單價分析用）
- `最接近比較標的.csv`（設定 `系爭房屋` 時產生，依接近度排序）
- `比較標的_排名NN.png`（**本工具核心產出**：對最接近的前 N 筆，各自在官方畫面把該列框紅截圖，可直接附卷）
- `操作紀錄.txt`（含截圖 SHA-256 雜湊、門牌彙總、最接近前 5 名、各截圖對應之比較標的）

> 註：結果表是分頁顯示（每頁 15 筆），本工具會自動逐頁點「下一頁」擷取**全部**。`擷取頁數` 設為 `"全部"` 或 `0` 即啟用；設為 `1` 則只抓第 1 頁。

## 設定欄位

- `縣市`：例如 `臺北市`
- `行政區`：例如 `中山區`（留空 `""` 則整個縣市搜尋）
- `門牌社區關鍵字`：填門牌或社區名稱可縮小範圍，例如 `民權東路二段71巷`。留空則不限。
- `交易期間`：民國年月
- `交易標的`：常用為 `["房地"]`
- `排除房地車`：建議設 `true`（見下方說明）
- `建物型態`：例如 `["公寓"]`；留空 `[]` 表示不限
- `屋齡`：可填 `不拘`、`0-5`、`5-10`、`10-20`、`20-30`、`30-40`、`40以上`
- `顯示瀏覽器`：`true` 會看得到瀏覽器操作，`false` 則背景執行

## 重要觀念

### 「排除房地車」是怎麼做的

官網「房地」這個交易標的，**本身就同時包含「純房地」與「房地(車位)」**，在首頁查詢階段無法分開。本工具的作法是：照官網查出全部，截圖與 `官方查詢結果第1頁.csv` 保留完整原貌（與畫面一致），另外依結果表「交易標的」欄的車位數，把含車位的成交濾掉，存成 `篩選後_排除房地車.csv` 供單價分析。為什麼要排除：含車位的成交，總價與面積都被車位墊高，算出的每坪單價會失真。

### 「最接近比較標的」怎麼排序

設定 `系爭房屋` 後，工具會依「接近度」排序比較標的，原則由近到遠：
同門牌（同棟）優先 → 同型態（以總樓層分組，避免老公寓與新大樓混比）→ 交易日越近越前。
品質懲罰由重到輕：無單價（全棟交易）→ 特殊/瑕疵交易、一樓 → 有增建（單價偏高）。
含車位者已剔除。各筆於 `最接近比較標的.csv` 的「註記」欄會標明其性質。

排好後，工具會對**最接近的前 `截圖最接近筆數` 筆**，逐一在官方結果頁把該列框紅、各截一張
`比較標的_排名NN.png`，這是可直接附卷的核心產出。排序與標記僅供初步篩選，
最終比較標的之取捨由法院認定。

設定範例：

```json
"系爭房屋": {
  "門牌關鍵字": "71巷27號",
  "樓層": 4,
  "建物總樓層": 4,
  "屋齡": 50,
  "起訴年月": "113/5",
  "主建物面積": null
},
"截圖最接近筆數": 5
```

### 用「社區名稱」常常查無資料

實價登錄的「社區簡稱」欄位政府**經常沒有登錄**，所以直接用社區名稱（例如「亞昕首藏」）搜尋，往往查無結果，即使該社區的成交確實存在。**較可靠的作法是用門牌/巷弄搜尋**（例如該社區所在的 `民權東路二段71巷`），就能抓到同棟、同巷的成交。

## 注意

官方網站若改版，欄位代號（DOM id）可能改變，屆時要調整 `scripts/official_lvr_evidence.js`。腳本內已用中文註解標出各欄位對應的 `name`／`id`（如 `ptype`、`f_type`、`avg_var`、`#p_build` 等）。

若卷內需要完整逐筆明細，可再擴充「逐筆點開明細並截圖」與「自動翻頁擷取全部頁次」功能。
