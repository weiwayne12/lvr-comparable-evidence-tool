# 專案知識庫（核心規範）

> 本檔為本專案 AI 協作規範的**唯一真實來源**。
> `CLAUDE.md`（Claude Code）與 `AGENTS.md`（Codex）皆指向本檔，請勿在那兩份重複撰寫內容，一律只改本檔。

## 專案用途

- 本專案是「實價登錄核定訴訟標的價額工具」。
- 用途：協助以官方實價登錄資料，產出查詢截圖與佐證資料。
- 截圖為主要依據，CSV/JSON 僅作輔助整理。

## 協作規範

- 全程使用繁體中文回覆。
- 法律內容、訴訟標的價額判斷、官方資料來源不得臆測；涉及法律或官方資料時，必須查證來源。
- 優先採用官方資料來源（如內政部不動產交易實價查詢服務網、司法院／全國法規資料庫）。
- 不得把終端機顯示的中文亂碼，直接判定為檔案損壞；本專案中文檔名與文件內容可能是正常 UTF-8。
- 編輯含中文的檔案時，請確保存成 **UTF-8（無 BOM）**，勿轉成 Big5 或加上 BOM。

## 專案慣例

- 案件設定檔慣例為 `案件設定*.json`。
- 輸出檔案慣例放在 `output/`。
- 驗證工具是否可執行時，使用 `npm run evidence`。

## 修改與驗證

- 修改查詢、自動化、截圖、輸出格式相關程式後，應盡可能實際執行 `npm run evidence` 驗證。
- 不要任意改動既有輸出檔名、欄位、資料夾結構，除非使用者明確要求。
